## Lecture 29

<table> is an html table
	<tr> for each row
		<td> for each element in the table

<thead> and <tbody> are optional tags to segment the table into a header and body. Otherwise the rows will all appear as body rows


> <table border="1">
> <thead>
> 	<tr>
> 		<th>Name</th>
> 		<th>Age</th>
> 		<th>LastRow</th>
> 	</tr>
> </thead>
> <tbody>
> 	<tr>
> 		<td>Red</td>
> 		<td>Orange</td>
> 		<td>Blue</td>
> 	</tr>
> 	<tr>
> 		<td>Red</td>
> 		<td>Orange</td>
> 		<td>Blue</td>
> 	</tr>
> </tbody>
> </table>
